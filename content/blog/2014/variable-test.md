一些blog中没有提到的：

发布草稿

```
$ hexo publish <filename>
```

缩写：hexo p

```
$ hexo clean
```
去除缓存`db.json` and 文件夹 `public`.
```
$ hexo server --debug
```
开启debug模式，所有更改会显示在命令行中并保存在根目录的debug.log文件中
在浏览器中可以看到即的变化
